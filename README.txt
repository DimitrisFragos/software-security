Installation Steps

1) Download and install XAMPP or LAMPP depending on your operating system 

------------------------> https://www.apachefriends.org/download.html

2) Install a database management tool like HeidiSQL or MySQL Workbench

------------------------>https://www.heidisql.com/

3) Copy the entire e-shop folder in your XAMPP htdocs folder

4) Activate Apache and MySQL from XAMPP

5) Open the database management tool 

6) Connect to Mysql running from XAMPP from the database management tool
Default credentials: username=root, pass=""

7) Create new Database with the name eshopdb

8) Load and execute the sql files found in the project folder: "cart_system.sql" and "test_db.sql"

9) Open your browser under the url localhost/eshop/login.php and you will find the login page to the eshop.
